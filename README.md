# book-recommendation-system
O sistema permitirá que usuários pesquisem livros, avaliem obras e recebam recomendações personalizadas. O mecanismo de recomendação deverá utilizar algoritmos de filtragem baseada em conteúdo (Content-Based Filtering) e, opcionalmente, filtragem colaborativa (Collaborative Filtering), possibilitando comparar diferentes abordagens.
